# プログラミング言語 AiScReam

歩夢ちゃん ポインタの指す値をインクリメント

四季ちゃん ポインタの指す値をデクリメント

ルビィちゃん ポインタを右に移動

アイスクリーム! ポインタを左に移動

はーい ポインタが指す値が 0 なら、対応する 何が好き? の直後にジャンプする

何が好き? ポインタが指す値が 0 でないなら、対応する はーい の直後にジャンプする

あ・な・た 入力から 1 バイト読み込む

叫びましょ ポインタの指す値を出力する
